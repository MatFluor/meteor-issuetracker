meteor-issuetracker
===================

An issue tracking web app built with Meteor

Setup
-------------------
1. Install Meteor
2. $ git clone https://github.com/nathan-j-fiedler/meteor-issuetracker.git
3. $ cd meteor-issuetracker
4. $ meteor
5. Open browser to http://localhost:3000
